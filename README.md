# sales-intelligence-powerbi-dashboard

This project focuses on building an interactive Power BI dashboard to analyze sales performance across products, customers, regions, channels, and warehouses.

## Tools Used
- Power BI
- Microsoft Excel / Google Sheets
- DAX

## Key Objectives
- Monitor key sales KPIs
- Analyze trends and seasonality
- Identify high-value customers
- Evaluate product, regional, and channel performance
